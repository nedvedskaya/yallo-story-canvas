# Яло — Редактор каруселей и сторис

## Концепция

Премиальный mobile-first редактор слайдов с glassmorphism эстетикой, вдохновлённый приложенным референсом. Акцент — #FFF1B6 (тёплый золотистый).

## Дизайн-система

- **Glassmorphism**: полупрозрачные панели с backdrop-blur, мягкие тени
- **Градиенты**: нежные переходы от голубовато-серого к тёплому бежевому (как на референсе)
- **Акцент**: #FFF1B6 для активных элементов, подсветки навигации, кнопок
- **Типографика**: минималистичная, тонкие начертания для премиального ощущения
- **Скруглённые углы**: крупные радиусы для карточек и панелей

## Структура экрана

### Верхняя панель (glassmorphism)

- Логотип «Яло» слева
- Кнопка «Скачать» справа (акцентный цвет)

### Центральная зона — Карусель слайдов

- Слайды расположены горизонтально друг за другом с отступами (как карусель Instagram)
- Горизонтальный скролл с snap-эффектом
- Активный слайд по центру, соседние видны по краям
- Индикатор текущего слайда (точки)
- Кнопки добавления/удаления слайда
- Каждый слайд — редактируемая область с фоном и текстовыми блоками

### Нижнее меню (glassmorphism, фиксированное)

5 иконок с подписями, активная подсвечена акцентным цветом:

1. **Дизайн** — выбор шаблонов/стилей слайда
2. **Фон** — выбор цвета/градиента/изображения фона
3. **Текст** — добавление и редактирование текста
4. **Размер** - размер слайдов 
5. **Инфо** — информация о проекте

## Интерактивность

- Свайп между слайдами с плавной анимацией
- Нажатие на нижнее меню открывает соответствующую панель (bottom sheet с glassmorphism)
- Редактирование текста по тапу на текстовый блок
- Смена фона с живым превью
