

# Исправить нижнее меню — настоящий эффект стекла

## Проблема
Сейчас меню имеет розовый фон `rgba(255, 220, 225, 0.35)` — это выглядит просто розовым, а не стеклянным. На фоне `#FDF6F0` розовый тон слишком заметен и не создаёт ощущения прозрачного стекла.

## Решение
Заменить розовый фон на нейтральный белый полупрозрачный, усилить blur и добавить более мягкую границу — как на референсе (Apple-style frosted glass):

**Файл:** `src/components/editor/BottomMenu.tsx`

Новые inline-стили контейнера:
- `background: rgba(255, 255, 255, 0.45)` — нейтральный белый, без розового оттенка
- `backdropFilter: blur(40px) saturate(180%)` — усиленный blur для матового стекла
- `border: 1px solid rgba(255, 255, 255, 0.6)` — тонкая белая граница
- `boxShadow: 0 4px 24px rgba(0, 0, 0, 0.06), inset 0 1px 0 rgba(255, 255, 255, 0.5)` — мягкая тень без розового

Цвета иконок/текста сделать нейтральными:
- Неактивные: `rgba(60, 60, 67, 0.45)` (серый, как в iOS)
- Активные: `rgba(60, 60, 67, 0.9)` (тёмно-серый)
- Индикатор активной вкладки: `rgba(60, 60, 67, 0.8)`
